Ziel/Grund zur Niederlage:

Wenn das Geld unter 50 fällt und kein Einkommen mehr da ist, ist das Spiel verloren (vorübergehend)

=====================================================

Statische Symbole:

§50🏠 und $250🏛️ produzieren nach jeder Runde einen gewissen Betrag (🏠=$10/🏛️=$75) an Geld/bringen Einkommen

$200🏭 Fabriken werden benötigt um 🚒 und 🚗 neben 🏭 platzieren zu können
$250🛫 Flüghäfen werden benötigt um 🛩️ neben 🛫 platzieren zu können

=====================================================

Dynamische Symbole:

$100🚗 Panzer können Gebäude (🏠🏛️🏭🛫) und Flags (🚒) zerstören
$150🚒 Flags können 🛩️ zerstören
$100🛩️ Flugzeuge können 🚗 zerstören

Fahrzeuge können Symbole zerstören indem man sie auf das entsprechende Feld bewegt (man kann auch seine eigenen Symbole zerstören)

=====================================================

Raketen:

1 kostet $300 und zerstört nur das beschossene Feld
2 kostet $2000 und zerstört Felder im 3x3 Umkreis
3 kostet $6000 und zerstört Felder im 5x5 Umkreis

ACHTUNG: ein beschossenes Feld ist nach dem Beschuss völlig unbrauchbar

=====================================================

Steuerung:

Objekte platzieren/Raketen abfeuern : Linksklick

Shop für Symbole öffnen : Rechtsklick

Shop für Raketen öffnen : b
	ACHTUNG: Rakete ist nach einmaligem auswählen gekauft und kann nicht "rückerstattet werden" wenn Linksklick gedrückt wird, wird das angeklickte Feld beschossen/wenn 	eine weitere Rakete vor dem Abschuss der vorherigen Rakete gekauft wird, wird die ungenutzte Rakete "gelöscht" und das ausgegebene Geld ist verloren

Fahrzeug auswählen : x

Fahrzeug steuern : Pfeiltasten
=====================================================
